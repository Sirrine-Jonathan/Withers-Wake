# Wither's Wake

A 2D gothic platformer built with the Kraken Engine for the "Decay" Game Jam.

## Theme: Decay
In a world lost to time and decay, you play as "The Bloom," an entity that can restore temporary life. Navigate through monochromatic ruins by blooming temporary platforms and collecting Life Sparks to sustain your essence.

## Controls
- **A / D**: Move Left / Right
- **Space**: Jump
- **Left Mouse Click**: Bloom a platform (costs Essence)
- **R**: Restart (after Win/Game Over)

## Setup
1. Create a virtual environment: `python -m venv venv`
2. Activate it: `venv\Scripts\activate` (Windows)
3. Install dependencies: `pip install kraken-engine`
4. Run the game: `python main.py`
